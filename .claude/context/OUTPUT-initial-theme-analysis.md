Run the onboard-theme skill once at the start of a project. AI will analyze the entire theme and document:

- **Theme overview** — Name, version, base theme (Dawn, Debut, custom, etc.)
- **Grid system & layout** — Wrapper classes, column grids, max-width, page-width patterns
- **Breakpoints** — Mobile, tablet, desktop breakpoint values
- **CSS conventions** — Naming patterns (BEM, utility, custom), color variables, spacing system
- **Typography** — Font stacks, heading styles, text utilities
- **Liquid patterns** — How sections, snippets, and templates are structured
- **Schema patterns** — Common setting types, block patterns, preset conventions
- **JavaScript patterns** — Existing web components, global JS, event systems, cart/drawer APIs
- **Existing components** — Modals, drawers, sliders, tabs, accordions, and other reusable UI already in the theme
